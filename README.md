# Antena Tec – Tema WordPress

![WordPress](https://img.shields.io/badge/WordPress-Compatible-blue)
![PHP](https://img.shields.io/badge/PHP-7.4%2B-purple)

Tema WordPress desenvolvido para o portal **Antena Tec**, focado na publicação de notícias e conteúdos relacionados à tecnologia, inovação e tendências digitais.

O projeto foi desenvolvido com foco em **desempenho, SEO e responsividade**, utilizando uma estrutura simples e eficiente baseada nos templates padrão do WordPress.

---

## 📌 Visão Geral

O tema **Antena Tec** foi criado para um portal de conteúdo tecnológico, permitindo a publicação de artigos, notícias e conteúdos informativos com layout moderno e otimizado para leitura.

O design foi pensado para oferecer:

- navegação simples
- carregamento rápido
- compatibilidade com dispositivos móveis
- boa indexação em mecanismos de busca

---

## 🚀 Principais Funcionalidades

- Layout responsivo
- Estrutura de templates padrão do WordPress
- Página inicial personalizada (`front-page.php`)
- Página de listagem de posts (`archive.php`)
- Página de busca (`search.php`)
- Página individual de artigos (`single.php`)
- Suporte a imagens para redes sociais
- Estrutura otimizada para SEO
- Arquitetura leve e de fácil manutenção

---

## 🛠 Tecnologias Utilizadas

- WordPress
- PHP
- HTML5
- CSS3
- JavaScript
- Tailwind CSS

---

## 📄 Templates Utilizados

O tema utiliza os principais templates do WordPress:

| Template | Função |
|--------|--------|
| `front-page.php` | Página inicial |
| `index.php` | Estrutura base do tema |
| `archive.php` | Listagem de posts |
| `single.php` | Página individual do artigo |
| `page.php` | Páginas institucionais |
| `search.php` | Página de resultados de busca |
| `header.php` | Cabeçalho do site |
| `footer.php` | Rodapé do site |

```bash
